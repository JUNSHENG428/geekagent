# Local Agent Wizard Media Kit

## 项目定位

Local Agent Wizard 是面向本地 AI Agent 用户的配置生成和教程站，帮助用户根据硬件、用途和安全偏好生成 Ollama、Cursor MCP、Qwen-Agent、Loop Engineering 配置包。

## 目标受众

- 本地 AI 新手
- Ollama / LM Studio 用户
- Cursor MCP 用户
- 独立开发者
- 内容创作者
- 想做企业私有化 AI 的团队

## 广告适配品类

- 本地 AI 硬件：Mac、显卡、迷你主机、内存、SSD
- 云 GPU / 推理服务
- AI IDE / Agent 工具
- MCP Server / 自动化工具
- 向量数据库 / RAG 工具
- 本地 AI 课程、咨询、模板包
