FAQ

常见问题

这个工具会上传我的数据吗？

不会。当前版本是纯客户端静态工具，所有推荐和模板生成都在浏览器里完成。

推荐结果一定准确吗？

它是基于规则表的保守估算。不同系统负载、上下文长度和后台应用都会影响实际表现。

为什么要留 30% 内存？

因为工具链、KV Cache、浏览器自动化、系统后台进程都会占内存。本地 Agent 要稳定，不能把内存全部给模型。

Cursor MCP 配置失败怎么办？

先检查 Node.js、npx、uvx 是否安装，再只保留 filesystem 做最小测试。也可以查看 Cursor MCP 教程。

可以接广告吗？

可以，但建议广告和用户需求强相关，例如本地 AI 硬件、云 GPU、AI IDE、课程和配置诊断服务。合作方式见 广告合作页。

数据库和插件应该先接什么？

建议先接 Supabase 和 Resend，先沉淀订阅、评论、配置生成和诊断线索。支付、分析、反垃圾插件可以按商业化阶段逐步接入，详见 插件与数据库方案。

付费模板包和免费工具有什么区别？

免费工具生成基础配置；高级模板包提供真实场景 Prompt、Loop Playbook、验收标准和故障排查清单。

可以预约配置诊断吗？

可以。适合已经遇到 Ollama、MCP、Python 环境问题，或者需要根据硬件做一套稳定方案的用户。见 定价页。
