Mac 如何搭建本地 AI Agent

Apple Silicon 的优势是统一内存和能效。关键不是选最大模型，而是让模型、工具链和系统占用保持平衡。

内存选择工具栈步骤硬件建议

内存怎么选

  • 8GB：只建议轻量聊天和简单工具调用。
  • 16GB：适合 Qwen3 8B Q4，能做文件处理和代码辅助。
  • 32GB：可以尝试 14B 模型和 RAG。
  • 64GB+：适合多工具 Agent、长上下文和更大模型。

推荐工具栈

新手优先选择 Ollama + Cursor MCP。需要 Python Agent 时再加入 Qwen-Agent。网页自动化任务再安装 Playwright。

brew install ollama node
brew services start ollama
ollama pull qwen3:8b

最快路径

  1. 打开 Local Agent Wizard。
  2. 设备选择 Mac，填入内存。
  3. 选择用途模板，例如代码审查助手或文件整理助手。
  4. 复制安装命令和 MCP 配置。
  5. 运行健康检查，确认 Ollama 和 MCP 都能工作。
建议先用小任务测试：列出工作目录文件，不要修改任何文件。确认工具链可用后，再让 Agent 写入或批量处理。
