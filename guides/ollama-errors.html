Troubleshooting

Ollama 常见报错解决合集

很多本地 Agent 失败不是模型问题，而是服务没启动、模型没下载或依赖缺失。

connection refused

人话解释：Ollama 服务没启动，或者端口不是 11434。

curl http://localhost:11434/api/tags
brew services start ollama
sudo systemctl start ollama

model not found

人话解释：配置里写了模型，但本机还没下载。

ollama list
ollama pull qwen3:8b

out of memory

人话解释：模型、上下文或工具链超过内存/显存预算。

  • 降级模型，例如从 14B 换成 8B。
  • 关闭浏览器、IDE、视频软件。
  • 先移除 embedding 或 playwright。

Agent 能聊天但不能调用工具

通常是 MCP server 没启动、command 不存在或路径不对。先只保留 filesystem 做最小测试。
