Model Guide

不同内存能跑什么本地 AI 模型？

本地 Agent 不只消耗模型内存，还要给操作系统、工具链和 KV Cache 留空间。

简单规则

可分配给模型的内存大约是：总内存减去系统占用后，再乘以 70%。剩余空间留给工具链和上下文。

推荐表

  • 小于 5GB：Qwen3 1.7B / Phi-4-mini，适合轻量对话。
  • 5-8GB：Qwen3 8B Q4_K_M，适合基础 Agent。
  • 8-12GB：Qwen2.5-Coder 14B Q4_K_M，适合代码辅助。
  • 12-20GB：Qwen3 14B / Mistral 22B Q4，适合复杂任务。
  • 20GB+：Qwen3 30B-A3B MoE，适合高质量推理。

不要只看模型大小

RAG embedding、Playwright 浏览器、代码解释器都会额外占内存。稳定性比参数量更重要。

如果你不确定，直接用 Wizard 生成保守方案，再逐步升级模型和工具链。
