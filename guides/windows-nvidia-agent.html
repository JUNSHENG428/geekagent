Windows + NVIDIA 如何搭建本地 AI Agent

Windows 用户最重要的是显存预算。显存越小，越要控制模型大小、上下文长度和工具链数量。

按显存选择模型

  • 8GB VRAM：Qwen3 1.7B / Qwen3 8B Q4，适合轻量 Agent。
  • 12GB VRAM：Qwen3 8B 或 Qwen2.5-Coder 14B Q4，适合代码辅助。
  • 16GB VRAM：14B 模型更稳，也可以加 Playwright。
  • 24GB+ VRAM：可尝试更大模型和多工具链。

安装顺序

  1. 安装 NVIDIA 驱动。
  2. 安装 Ollama Windows 版。
  3. 安装 Python 3.10+ 和 Node.js。
  4. 用 Wizard 生成配置。
ollama pull qwen3:8b
pip install -U "qwen-agent[mcp,gui,rag,code_interpreter]"
npx --version

常见问题

如果出现 CUDA out of memory，先降级模型，再减少工具链。网页自动化和 embedding 都会增加内存压力。
