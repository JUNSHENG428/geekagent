Cursor MCP 配置教程

MCP 让 Cursor 可以调用本地文件系统、网页抓取、数据库和记忆工具。配置正确后，Agent 才能真正处理本地任务。

配置步骤

  1. 打开 配置生成器。
  2. 目标工具选择 Cursor IDE。
  3. 复制 MCP 配置 Tab 中的 JSON。
  4. 打开 Cursor 设置中的 MCP 配置入口。
  5. 粘贴并保存，然后重启或刷新 MCP。

最小配置示例

{
  "mcpServers": {
    "filesystem": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "./workspace"]
    }
  }
}

验证方式

先让 Cursor 执行低风险任务：列出工作目录文件。不要一开始就让它删除、覆盖或批量移动文件。

如果看到 spawn npx ENOENT，通常是 Node.js 没装。先安装 Node，再重试。
