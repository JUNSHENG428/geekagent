🎉 你的 Agent 配置包

复制下方内容，粘贴进对应工具即可开始。

ℹ️ 硬件: M5 · 16GB · 推荐模型: Qwen3 8B Q4_K_M · 工具链: filesystem + memory
PYTHON — agent.py 

          

        

        

          💡
          将此文件保存为 agent.py，运行 python agent.py 即可启动。

          确保 Ollama 已启动并加载了对应模型。
        

        

          使用说明
          
先运行安装命令并拉取模型，再把此文件放到你的工作目录。首次运行建议用简单任务测试工具链是否能正常连接。

        

      


      
      

        

          

            JSON — Cursor / Claude Desktop settings
            
          

          

            

          

        

        

          📍
          Cursor: 打开 Settings → MCP，粘贴此 JSON。

          Claude Desktop: 粘贴进 ~/.claude/settings.json 的 mcpServers 字段。
        

        

          使用说明
          
建议先只启用 filesystem 和 fetch 等低风险工具，确认路径和权限正确后再添加 playwright、embedding 等资源消耗更高的工具。

        

      


      
      

        

          

            PROMPT — System Prompt
            
          

          

            

          

        

        

          ✏️
          将 [YOUR_WORKSPACE_PATH] 替换为你的工作目录路径（例如 /Users/你的用户名/workspace）。
        

        

          使用说明
          
如果你选择了较高自主度，仍建议保留删除、覆盖、联网提交等动作的确认规则，避免本地 Agent 误操作。
