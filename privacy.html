隐私政策

Local Agent Wizard 的核心原则是：本地计算、最少收集、清楚说明。

我们收集什么？

默认情况下，我们不会上传你的硬件配置、用途选择、生成的 Prompt 或配置文件。

浏览器本地处理

模型推荐、内存预算、MCP JSON、安装命令和 ZIP 打包都在你的浏览器中完成。

访问统计

我们可能统计页面访问量、来源和按钮点击，用来判断哪些教程和工具更有帮助。统计不会包含你生成的配置内容。

你主动提交的信息

如果你订阅更新、留言、预约诊断或购买模板，我们只会使用你主动提交的信息来提供对应服务。排错时请不要发送 API Key、密码、私密文件内容或未打码的完整日志。

广告和联盟链接

未来可能加入赞助推荐或联盟链接。我们会尽量保持广告和本地 AI 主题相关，并明确标注广告位。

免责声明

生成的命令和配置仅供参考。执行任何安装、文件写入、删除、自动化操作前，请先确认路径和权限。
