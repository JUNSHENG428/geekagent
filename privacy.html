隐私政策

Local Agent Wizard 的核心原则是：本地计算、最少收集、清楚说明。

我们收集什么？

当前静态版本不收集、不上传你的硬件配置、用途选择、生成的 Prompt 或配置文件。

浏览器本地处理

模型推荐、内存预算、MCP JSON、安装命令和 ZIP 打包都在你的浏览器中完成。

未来可能加入的统计

上线后可能使用 Plausible、Umami 或 Cloudflare Web Analytics 统计匿名访问量、页面来源和点击事件，用于改进产品。不会收集生成内容。

未来数据库和邮件功能

如果后续接入 Supabase、Resend、支付或评论插件，我们会把订阅、评论、诊断预约、购买记录等明确标注为用户主动提交的数据，并在表单附近说明用途。匿名配置统计会避免保存 API Key、文件路径、完整报错日志等敏感内容。

广告和联盟链接

未来可能加入赞助推荐或联盟链接。我们会尽量保持广告和本地 AI 主题相关，并明确标注广告位。

免责声明

生成的命令和配置仅供参考。执行任何安装、文件写入、删除、自动化操作前，请先确认路径和权限。
